# 🏥 Hospital Data Analysis using SQL

## 📌 Overview

This project analyzes hospital data to extract meaningful insights related to patient count, hospital performance, medical expenses, and treatment trends using SQL.

---

## 🎯 Objective

* Analyze patient distribution across hospitals and departments
* Identify high-performing hospitals
* Track medical expenses trends
* Understand patient stay patterns

---

## 🗂️ Dataset

* File: `Hospital_Data.csv`
* Contains:

  * Patient details
  * Hospital names
  * Departments
  * Admission & discharge dates
  * Medical expenses

---

## 🛠️ Skills Used

* SQL Aggregations (SUM, AVG)
* GROUP BY & ORDER BY
* Date Functions (DATEDIFF, YEAR, MONTH)
* Filtering (WHERE conditions)
* Ranking using TOP

---

## 📊 Key Analysis Performed

* Total number of patients
* Average doctors per hospital
* Top 3 departments by patient count
* Hospital with highest expenses
* Daily average medical expenses
* Longest hospital stay
* City-wise patient distribution
* Average stay per department
* Least busy department
* Monthly expense trends

---

## 📈 Key Insights

* Identified departments with highest patient load
* Found hospitals with maximum spending
* Analyzed patient stay duration patterns
* Tracked monthly expense growth trends
